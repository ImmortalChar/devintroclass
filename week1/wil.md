github를 사용하기 위한 git 설치, 편한 코드 수정을 위한 vs code 설치
원하는 폴더와 github repository 연결: github 페이지에서 command line 써준 것 복붙하여 가능! 중간에 로그인 필요함
폴더에 원하는 파일을 추가하고 다음 명령어 사용!
git status: repository와 현재 파일이 어떻게 다른지 확인
git add <file> : 해당 파일을 추가 또는 이미 있는 파일이면 수정된 것 반영
git commit -m "메세지" : 이 수정에 대해서 코멘트 달기
git push: 수정내용을 repository에 반영
단, 이때 repository에만 있고 폴더에 없는 파일의 경우 삭제되는 것이 아님!!
html 태그:
h1, h2... :헤드 태그 (글자 크기가 큼, 큰 숫자를 쓸 수록 글자 크기가 작아짐)
p : 텍스트
(추가로 공부한 img 태그)
img: 이미지 달기, 추가로 <> 안에 다음 사용 필요함
src = "파일명" 같은 폴더에 있으면 경로 설정이 필요 없지만 그게 아니면 경로 설정 필요
width = "" / height = "" 원래 크기로 쓸거면 무관하나 원하는 사이즈로 조정할 경우 필요
