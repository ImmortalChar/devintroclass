Github Desktop 프로그램: 깃허브 Repository에 있는 파일을 가져오고 업로드하는 데 훨씬 편해짐.
원하는 Repository를 선택하면 pc에 해당 파일을 가져올 수 있고, 그렇게 생성된 폴더에서 수정을 하면 그대로 업로드하는 기능이 있음.
하이퍼텍스트: <a href = "원하는.html" >원하는 텍스트</a> 형식으로 사용함
스타일 입히기: 텍스트 태그 중 앞의 것의 > 앞에 적어서 사용함. style = "원하는 것" 형식.
내부에는 color = 원하는 색(vs code에서 색 띄워주니 선택) >> 텍스트 색
background-color = 원하는 색 >> 텍스트 배경색 등이 있음.
